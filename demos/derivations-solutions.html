Three Roads to the Same x(t)

The lessons taught the ideas; this companion does the algebra. We take the one system this whole course revolves around — the mass–spring–damper — and solve it three independent ways: classically by hand, with the Laplace transform, and through the state-space matrix exponential. All three land on the identical closed-form motion x(t). Every step here was derived and cross-checked symbolically with SymPy.

The running example

One equation, in normalized form

Start from Newton's law for the mass–spring–damper (Lesson 2) and divide through by m. Writing the natural frequency ωₙ = √(k/m) and damping ratio ζ = c/(2√(km)) collapses the parameters into two meaningful numbers:

m ẍ + c ẋ + k x = F   ⟹   ẍ + 2ζωₙ ẋ + ωₙ² x = ωₙ²·xss   // xss = F/k is the steady-state rest position

We solve two canonical cases: the free response (F = 0, released from x(0)=x₀ with velocity ẋ(0)=v₀), and the step response (a suddenly applied constant force, starting from rest). Throughout, two shorthands recur: the decay rate σ = ζωₙ and the damped frequency ωd = ωₙ√(1 − ζ²).

Route 1 · Classical

Guess an exponential, match the initial conditions

Route 1 — solve the ODE directly
1
Homogeneous equation & the characteristic guess. With F = 0, try x = eλt. Every derivative just multiplies by λ, so the whole ODE reduces to a polynomial in λ:
ẍ + 2ζωₙ ẋ + ωₙ² x = 0   ⟹   λ² + 2ζωₙ λ + ωₙ² = 0   // the characteristic equation
2
Roots. The quadratic formula gives the two λ that make eλt a solution:
λ = −ζωₙ ± ωₙ√(ζ² − 1)
These are the same numbers the course calls the poles — and, in Route 3, the eigenvalues of A. Their nature depends entirely on ζ:
Underdamped 0≤ζ<1
λ = −σ ± jωd
xh = e−σt(A cos ωdt + B sin ωdt)
Critically damped ζ=1
λ = −ωₙ (repeated)
xh = (A + B t) e−ωₙt
Overdamped ζ>1
λ = −s₁, −s₂ (real)
xh = A e−s₁t + B e−s₂t
3
Apply the initial conditions (underdamped case). Impose x(0) = x₀ and ẋ(0) = v₀ on xh = e−σt(A cos ωdt + B sin ωdt). The first gives A directly; differentiating and setting t = 0 gives B:
A = x₀ ,    B = v₀ + σx₀ωd
4
The free response. Substituting back:
x(t) = e−σt [ x₀ cos ωdt + v₀ + σx₀ωd sin ωdt ]
A decaying envelope e−σt wrapped around an oscillation at ωd — exactly the ring-down the demos show.
5
The step response (start from rest, x₀ = v₀ = 0). Add the particular solution xp = xss (the new rest point) to the homogeneous part and re-apply the rest conditions:
x(t) = xss [ 1 − e−σt( cos ωdt + σωd sin ωdt ) ]
The classic overshoot-and-settle. SymPy's dsolve returns this exactly.

Route 2 · Laplace transform

Turn the ODE into algebra, then invert

Route 2 — transform, solve, invert
1
The tool. The transform X(s) = ∫₀∞ x(t) e−st dt has one property that matters here — differentiation becomes multiplication by s, carrying the initial conditions along:
L{ẋ} = s X(s) − x₀     L{ẍ} = s² X(s) − s x₀ − v₀
2
A small transform table (the only pairs we need — all verified with SymPy):

3
Transform the whole equation (free response) and collect X(s):
(s²X − s x₀ − v₀) + 2ζωₙ(sX − x₀) + ωₙ²X = 0
X(s) = s x₀ + v₀ + 2ζωₙ x₀s² + 2ζωₙ s + ωₙ²   // the denominator is the characteristic polynomial again
4
Complete the square in the denominator, s² + 2ζωₙs + ωₙ² = (s+σ)² + ωd², and split the numerator so each piece matches a table row:
X(s) = x₀ · s + σ(s+σ)² + ωd²  +  v₀ + σx₀ωd · ωd(s+σ)² + ωd²
5
Invert term by term using the last two table rows:
x(t) = e−σt [ x₀ cos ωdt + v₀ + σx₀ωd sin ωdt ]
Same answer. Route 2 reproduces Route 1 exactly — the Laplace method just trades hand-integration for table lookup. For the step input, X(s) = ωₙ²xss / [ s((s+σ)²+ωd²) ] partial-fractions to xss/s − xss(s+2σ)/((s+σ)²+ωd²), which inverts to the same step response as Route 1.

Route 3 · State-space

The matrix exponential

Route 3 — eAt x(0)
1
First-order form. With state x = [x, ẋ]ᵀ (Lesson 7), the second-order ODE becomes ẋ = Ax:
A = [ [ 0, 1 ], [ −ωₙ², −2ζωₙ ] ]   // characteristic polynomial of A: λ²+2ζωₙλ+ωₙ² — the same one
2
The solution is the matrix exponential. Exactly as the scalar equation ẋ = ax has solution x = eatx₀, the vector equation has:
x(t) = eAt x(0)     where   eAt = V eΛt V⁻¹   // eigen-decompose A = VΛV⁻¹; the eigenvalues in Λ are the poles
3
Evaluated (underdamped), the matrix exponential is:
eAt = e−σt
cos ωdt + (σ/ωd) sin ωdt(1/ωd) sin ωdt
−(ωₙ²/ωd) sin ωdtcos ωdt − (σ/ωd) sin ωdt
4
Read off the position — the top row of eAt times [x₀, v₀]ᵀ:
x(t) = e−σt [ x₀ cos ωdt + σx₀ + v₀ωd sin ωdt ]
Same answer, a third time. The matrix exponential's first row reproduces the exact x(t) of Routes 1 and 2. The velocity ẋ(t) falls out of the second row for free — a bonus the scalar methods don't hand you directly.

Key takeaways

  • The classical method guesses eλt, solves the characteristic equation, splits into three damping cases, and fits the initial conditions.
  • The Laplace method turns the ODE into algebra in s, solves for X(s), completes the square / partial-fractions, and inverts with a small table.
  • The state-space method writes ẋ = Ax and applies the matrix exponential x(t) = eAtx(0), giving position and velocity together.
  • All three share the same λ's — characteristic roots, transfer-function poles, and eigenvalues of A are one set of numbers.
  • All three yield the identical x(t), verified symbolically and numerically. Different bookkeeping, one physical motion.

Exercises

  1. By hand. For ωₙ = 2, ζ = 0.5, released from x₀ = 1, v₀ = 0, write σ, ωd, and the closed-form x(t).
  2. Laplace. Write X(s) for that same initial-value problem and identify its two poles.
  3. State-space. Write A for ωₙ = 2, ζ = 0.5 and confirm its eigenvalues equal the poles from Exercise 2.
  4. Concept. Explain why all three methods must give the same x(t), referring to what the roots/poles/eigenvalues have in common.
Show worked solutions

1. σ = ζωₙ = 0.5·2 = 1; ωd = ωₙ√(1−ζ²) = 2·√0.75 ≈ 1.732. With v₀ = 0, B = σx₀/ωd = 1/1.732 ≈ 0.577, so x(t) = e−t(cos 1.732t + 0.577 sin 1.732t).

2. X(s) = (s·1 + 0 + 2·0.5·2·1)/(s² + 2·0.5·2·s + 4) = (s + 2)/(s² + 2s + 4). Poles solve s² + 2s + 4 = 0 → s = −1 ± j√3 (i.e. −σ ± jωd).

3. A = [[0, 1], [−4, −2]]. Its characteristic equation is λ² − (trace)λ + det = λ² + 2λ + 4 = 0, so eigenvalues −1 ± j√3 — identical to the poles in Exercise 2.

4. Each method is just a different way of solving the same linear ODE, and each locates the system's natural frequencies as the roots of the same polynomial λ² + 2ζωₙλ + ωₙ² = 0 — called the characteristic roots (classical), the poles (Laplace/transfer function), or the eigenvalues of A (state-space). Since the governing equation, the initial conditions, and those defining roots are all identical, the solution x(t) they produce must be identical too. The methods differ only in bookkeeping, not in physics.
