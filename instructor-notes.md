# Instructor Notes

**Intended use.** A 5–7 hour intuition-first on-ramp before a full controls or fluid-power
course. Each lesson stands alone for ~25–40 minutes.

**Pacing.** Lessons 1–3 build physical modeling; 4–6 develop the frequency-domain view;
7–8 unify everything via state space and cross-domain analogies. Projects consolidate.

**Assessment.** Each lesson carries a conceptual quiz (5 Qs), calculation exercises (3),
and one design question, all with full worked solutions in dropdowns.

**Widgets.** Interactive sliders require a live kernel. Static plots are always shown
alongside so nothing breaks in the read-only website.
